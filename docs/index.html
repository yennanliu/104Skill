Frequently Asked Questions

Is this legal to use?

Yes! This tool is for personal productivity and educational purposes. Always use it responsibly and only apply to jobs you're genuinely interested in and qualified for.

How many jobs can I apply to?

We recommend applying to 10-20 jobs per session. Quality is more important than quantity. The skill includes safety features like rate limiting and delays.

Will 104.com.tw detect this?

The skill uses random delays (2-4 seconds by default) and respects rate limits. It mimics human behavior to avoid detection. However, always use responsibly.

Can I customize the cover letter?

Yes! You can specify different cover letters for different job types. Just configure the coverLetter option when running the skill.

What if something goes wrong?

The skill has built-in error handling. If a job application fails, it logs the error and continues with the next job. You can review the logs to see what happened.

Do I need to stay logged in?

Yes, you need to be logged in to 104.com.tw before running the automation. Keep your browser open during the process.
